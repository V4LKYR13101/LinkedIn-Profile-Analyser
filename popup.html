This extension enhances LinkedIn profiles with AI-powered analysis and interactive Q&A, supporting Gemma, Gemini, and Emma models.

To get a Google AI API key, visit Google AI Gemma

How to Use:

  1. Enter your API key above and select your preferred model
  2. Click "Save Settings"
  3. Visit any LinkedIn profile page
  4. Click the floating question mark button that appears
  5. Use "Analyze Profile" for instant insights
  6. Ask specific questions about the profile

Note: This extension only works on LinkedIn profile pages. It extracts profile information to help you gain insights through AI analysis.
